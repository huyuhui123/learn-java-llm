# 🎯 打破动力陷阱 · 晨间启动指南

- **你不是懒惰**，只是陷入了大脑的自我保护机制
- **选择放松是本能**，但我们可以巧妙引导这个本能
- **关键不是靠意志力硬撑**，而是重新设计行动流程

## 🌞 晨间启动五步法
1. **物理隔离**：睡前手机放远处，切断自动刷屏
2. **微小问题**：醒后问自己“今天能为职业做的最小一步是什么？”
3. **十分钟承诺**：“只做10分钟，完成后可以继续放松”
4. **动作连锁**：起床 → 洗漱 → 喝水 → 完成1个小任务 → 奖励自己
5. **进度可视化**：完成小任务后，在床头图表上打勾标记

## 🔄 困难与放松新关系
### 拆解任务
- 不是“完善简历” → 而是“更新一个项目描述”
- 不是“系统学习” → 而是“读一篇技术文章”
- 不是“找到工作” → 而是“浏览3个岗位”

### 建立配额
- 完成15分钟职业任务 = 获得30分钟视频时间
- 更新一段简历 = 阅读一章小说
- 投出一份简历 = 享受一集动漫
